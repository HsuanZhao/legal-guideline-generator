# 📋 法律业务操作指引生成器 (Legal Guideline Generator)

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

## 📖 简介

法律业务操作指引生成器是一个专门为律师和法律从业者设计的智能提示工程工具。通过模拟多位专业律师（诉讼律师、非诉律师、法官、学者）的集体智慧，整合上海高院公开判例、最高人民法院司法解释、指导案例、权威法律文章等可信信息源，自动生成《律师办理xxx业务操作指引》文档。

## ✨ 核心功能

- 🧑‍⚖️ **多律师视角模拟**：融合诉讼律师、非诉律师、法官、学者四种视角，提供全面的实务建议
- 📚 **权威信息整合**：优先使用最高人民法院司法解释、指导案例、上海高院判例、权威法律数据库
- 📐 **结构化输出**：遵循律协业务指引标准模板（总则、业务概述、业务流程、风险防范、争议解决、附则）
- 🎯 **业务类型适配**：针对侵权纠纷、合同纠纷、婚姻家事、非诉业务、刑事辩护等不同领域自动调整输出重点
- ⏱️ **时效性保障**：优先使用近2年案例和最新法律法规（如《新公司法》《民法典合同编解释》）

## 🎯 适用场景

- 👥 律师团队标准化办案流程
- 🧑‍🎓 法律实习生快速熟悉业务领域

## 🛠️ 使用方法

### 1. 在支持自定义指令的 AI 平台中使用

将 `SKILL.md` 文件内容复制到以下平台的系统提示或自定义指令中：

- **Claude.ai**（Projects 或 Custom Instructions）
- **ChatGPT**（Custom Instructions 或 GPT Builder）
- **本地部署的 LLM**（如 Ollama、LM Studio）
- **Cursor / Windsurf** 等 AI 编程助手

### 2. 触发生成指引

使用以下触发词之一，后接您的业务领域：
- 生成指引 艺人肖像网络侵权责任纠纷
- 办理指引 申请确认无民事行为能力加指定监护
- 业务指引 私募基金合同纠纷诉讼
- 律师操作指引 数据出境安全评估


### 3. 输入格式示例

- 生成律师办理艺人肖像网络侵权责任纠纷业务操作指引

或提供更详细的需求：
- 办理企业商业秘密保护合规体系建设业务操作指引，重点关注员工离职管理和竞业限制

## 📄 输出示例

生成的指引将包含以下章节：

- **第一章 总则**：目的依据、适用范围、基本原则
- **第二章 业务概述**：概念界定、业务分类、法律关系分析
- **第三章 业务流程**：接受委托 → 事实梳理与证据收集 → 法律分析 → 方案制定 → 代理操作 → 结案归档
- **第四章 风险防范**：常见风险点、防控措施、执业风险提示
- **第五章 争议解决**：管辖确定、程序问题、执行要点
- **第六章 附则**：法律依据、术语解释、生效日期

每个章节均包含具体法条引用、判例来源和实务要点提示（标注 `【律师工作提示】` 或 `【风险提示】`）。

## 📊 信息源优先级

| 优先级 | 来源 | 说明 |
|--------|------|------|
| 1 | 上海律协已发布业务指引 | 参考结构和表述方式 |
| 2 | 最高人民法院司法解释 | 法律依据和裁判要点 |
| 3 | 最高人民法院指导案例 | 同类案件裁判规则 |
| 4 | 上海高院公开判例 | 本地司法实践 |
| 5 | 权威法律公众号专业文章 | 时效性强，实务经验 |
| 6 | 权威法律文章/著作 | 学术观点和实务经验 |

## 📦 依赖与要求

- 🧠 需要 AI 模型具备足够的上下文长度（建议 8K tokens 以上，以生成长指引）
- 🌐 建议配合联网搜索功能使用，以获取最新判例和法规
- 🔌 无其他外部依赖

## ⚠️ 局限性声明

- 本工具生成的指引基于公开信息源，不保证涵盖所有实务情况
- 具体案件应当根据实际情况调整，不得直接替代律师专业判断
- 法律变化可能导致部分内容过时，使用者应自行核实最新法律法规
- 本工具不提供针对具体案件的法律意见
